🏋️‍♂️ API Academia

Sistema web moderno que simula um terminal de autenticação de academia, com entrada de CPF via teclado numérico interativo.

Vercel: https://api-academia-front-m8xy3bw5z-joaquims-projects-be9a7ecb.vercel.app/

📌 Sobre o Projeto

O API Academia foi desenvolvido com foco em interface e experiência do usuário, simulando um sistema real de entrada em academias.

A aplicação permite que o usuário insira seu CPF em um teclado digital estilizado, com validação visual e interações intuitivas.

💡 Ideal para projetos front-end, portfólio e futuras integrações com APIs reais.

🎯 Funcionalidades

✔️ Entrada de CPF via teclado numérico
✔️ Máscara automática (000.000.000-00)
✔️ Botão Limpar para resetar dados
✔️ Botão OK para envio
✔️ Interface moderna estilo terminal
✔️ Design responsivo e intuitivo
